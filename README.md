# Background Remove

A simple Python project to remove background from images using AI.

## 🚀 Features
- Remove background from images
- Simple and easy to use
- Fast processing

## 🛠️ Tech Stack
- Python
- Image Processing / AI

## 📂 Project Structure
main.py        # Main script  
requirements.txt  
README.md  

## ⚙️ Installation

1. Clone the repository
git clone https://github.com/Sheth3433/background_remove

2. Go to project folder
cd background_remove

3. Install dependencies
pip install -r requirements.txt

## ▶️ Usage

Run the main file:
python main.py

## 📌 Output
- Generates image with background removed
